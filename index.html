  • AI Intergration via API

    This app leverages Open Ai's Powerful DALLE API and Chat GPT 3 to generate comic book panes with relative captions. To challenge the functionality, user input is taken in the form of a madlib, which is generated from 4 hardcoded stories, and fed to the DALL-E API and Chat GPT to concurrently generate an illustration and caption based on user input. One of the major advantages of utilizing AI is that virtually any input even empty input can be ingested and produce a coherent illustration and caption. In addition, a seemingly infinite variation of stories and pictures can be generated off a single story format with the same inputs.

    The site itself demonstrates a practicle use case of mostly vanilla JavaScript (with some jquery) to call API functions to give the site the ability to complete more complicated tasks. The site does not rely on a remote server for API authentication or use .env files to hide API keys. A simple workaround is put in place where the user is instructed to create a new API key if the site does not detect an API key in local memory. I can't wait for you to try the site out!
